ZZ - Arbitrary-Precision Integers

LibNTL.ZZ — Type
ZZ

Arbitrary-precision integer type. In development mode, this is a wrapper around BigInt.

source
LibNTL.numbits — Function
numbits(z::ZZ) -> Int

Return the number of bits in the binary representation of |z|. Returns 0 for z = 0.

source
LibNTL.numbytes — Function
numbytes(z::ZZ) -> Int

Return the number of bytes needed to represent |z|. Returns 0 for z = 0.

source

Arithmetic

ZZ supports all standard Julia arithmetic operators:

  • +, -, *: Addition, subtraction, multiplication
  • ^: Power (non-negative integer exponents)
  • div, rem, mod, divrem: Integer division operations
  • gcd, gcdx: GCD and extended GCD

Conversion

# From various types
ZZ(42)                    # From Int
ZZ("12345678901234567890")  # From String
ZZ(big"999999999999")     # From BigInt

# To other types
BigInt(ZZ(42))            # To BigInt
convert(Int64, ZZ(42))    # To Int64 (throws if overflow)
string(ZZ(42))            # To String

Predicates

  • iszero(z): Check if zero
  • isone(z): Check if one
  • isodd(z), iseven(z): Parity checks
  • sign(z): Returns -1, 0, or 1

Modulus Management

Before using ZZ_p, you must set a modulus:

ZZ_p_init!(ZZ(17))  # All operations are now mod 17

For temporary modulus changes:

with_modulus(ZZ(23)) do
    # Operations here use modulus 23
    a = ZZ_p(20)
    println(rep(a))  # 20
end
# Original modulus is restored

Arithmetic

ZZ_p supports:

  • +, -, *: Field operations
  • /: Division (via multiplicative inverse)
  • ^: Power
  • inv: Multiplicative inverse

ZZ_pContext

LibNTL.ZZ_pContext — Type
ZZ_pContext

Context for saving/restoring ZZ_p modulus state.

source

For manual context management:

ctx = ZZ_pContext()
save!(ctx)        # Save current modulus
ZZ_p_init!(ZZ(31))  # Change modulus
# ... work with modulus 31 ...
restore!(ctx)     # Restore previous modulus

LibNTL.primpart — Function
primpart(f::ZZX) -> ZZX

Compute the primitive part (f / content(f)) of f.

source

Construction

# From coefficient vector [a_0, a_1, ..., a_n]
f = ZZX([ZZ(1), ZZ(2), ZZ(3)])  # 1 + 2x + 3x^2

# From scalar (constant polynomial)
g = ZZX(ZZ(5))  # 5

# Zero polynomial
h = ZZX()

Coefficient Access

f = ZZX([ZZ(1), ZZ(2), ZZ(3)])
degree(f)     # 2
f[0]          # ZZ(1) - constant term
f[2]          # ZZ(3) - leading coefficient
leading(f)    # ZZ(3)
constant(f)   # ZZ(1)

Arithmetic

  • +, -, *: Polynomial arithmetic
  • c * f, f * c: Scalar multiplication
  • div, rem, divrem: Polynomial division
  • gcd: Polynomial GCD

Polynomial Operations

f = ZZX([ZZ(1), ZZ(2), ZZ(3)])

# Evaluation
f(ZZ(5))  # Evaluate at x = 5

# Derivative
derivative(f)  # 2 + 6x

# Content and primitive part
g = ZZX([ZZ(6), ZZ(12), ZZ(18)])
content(g)   # ZZ(6)
primpart(g)  # 1 + 2x + 3x^2

Iteration

ZZX supports iteration over coefficients:

f = ZZX([ZZ(1), ZZ(2), ZZ(3)])
for c in f
    println(c)
end
# Output: 1, 2, 3

Number Theory Functions

LibNTL.PowerMod — Function
PowerMod(a::ZZ, e::ZZ, n::ZZ) -> ZZ

Compute a^e mod n using binary exponentiation.

Arguments

  • a: Base
  • e: Exponent (can be negative if a is invertible mod n)
  • n: Modulus (must be > 1)

Returns

  • Result in range [0, n-1]

Examples

PowerMod(ZZ(2), ZZ(10), ZZ(1000))  # 2^10 mod 1000 = 24
PowerMod(ZZ(3), ZZ(-1), ZZ(7))     # 3^(-1) mod 7 = 5

Errors

  • Throws DomainError if n <= 1

LibNTL.ProbPrime — Function
ProbPrime(n::ZZ, num_trials::Int=10) -> Bool

Probabilistic primality test using Miller-Rabin.

Arguments

  • n: Number to test
  • num_trials: Number of Miller-Rabin iterations (default: 10)

Returns

  • true if n is probably prime
  • false if n is definitely composite

Examples

Polynomial Functions

LibNTL.factor — Function
factor(f::ZZX) -> (ZZ, Vector{Tuple{ZZX, Int}})

Factor the polynomial f over the integers.

Returns a tuple (content, factors) where:

  • content is the content of f (GCD of all coefficients)
  • factors is a vector of (factor, multiplicity) pairs

The product of content times all factors raised to their multiplicities equals f.

Example

f = ZZX([ZZ(-1), ZZ(0), ZZ(1)])  # x^2 - 1
c, factors = factor(f)
# c = 1, factors = [(x-1, 1), (x+1, 1)]
source
LibNTL.cyclotomic — Function
cyclotomic(n::Integer) -> ZZX

Compute the n-th cyclotomic polynomial Φₙ(x).

The cyclotomic polynomial Φₙ(x) is the minimal polynomial of primitive n-th roots of unity over the rationals. It satisfies:

  • xⁿ - 1 = ∏{d|n} Φd(x)
  • degree(Φₙ) = φ(n) (Euler's totient function)

Examples

cyclotomic(1)   # x - 1
cyclotomic(2)   # x + 1
cyclotomic(3)   # x² + x + 1
cyclotomic(4)   # x² + 1
cyclotomic(6)   # x² - x + 1
source

InvModError

LibNTL.InvModError — Type
InvModError(a::Any, n::Any)

Exception thrown when computing a modular inverse fails because gcd(a, n) ≠ 1.

Fields

  • a: The value that failed to be inverted
  • n: The modulus

Example

try
    inv(ZZ_p(0))
catch e::InvModError
    println("Cannot invert ", e.a, " mod ", e.n)
end
source

Thrown when computing a modular inverse fails:

ZZ_p_init!(ZZ(10))
try
    inv(ZZ_p(5))  # 5 has no inverse mod 10
catch e::InvModError
    println("Cannot invert ", e.a, " mod ", e.n)
end
